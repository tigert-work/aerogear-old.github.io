AeroGear Push Module

Send push notifications to any device, regardless of platform or network.

The push module consists of the AeroGear Unified Push Server which sends native messages to different mobile devices.

In addition to the Unified Push Server there are two additional backend modules: SimplePush Server that uses WebSockets, intended for WebOS, but due to its light weight would also work for small devices like Internet of Things applications.

There is also work underway to implement a server based on the HTTP2 WebPush specification, which will likely be the successor of SimplePush service in the future.
