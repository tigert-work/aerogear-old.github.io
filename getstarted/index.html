Getting Started with AeroGear

AeroGear consists of a backend service in OpenShift cloud platform, and client libraries for various mobile platforms.

Follow the tutorials linked below to get everything set up, installed and build your first mobile application using AeroGear.

The links will open in a new browser window, so after completing each step, please return to this page and continue where you left off.

1. Spin up the Backend Service on OpenShift.

Follow our Quickstart Guide to set up your backend. The AeroGear Backend Service will be the middle man between your mobile application and your data.

2. Install the Mobile SDK of your choice

If you already have your mobile application development tools set up, you can skip this, and just install the AeroGear libraries, as instructed in the next step.

However, in case you are new to mobile app development in general, select your targeted mobile platform below, and follow our instructions to get everything set up.
