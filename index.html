Multi-Platform Mobile Toolkit

The Aerogear Project provides flexible, extensible libraries and integrated server-side components that will simplify your mobile development and infrastructure setup across multiple mobile platforms.

If you are new to AeroGear, we suggest you walk through our Getting Started tutorial first, to get everything set up for development. You can also learn more about each module below.
