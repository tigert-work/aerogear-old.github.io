Multi-Platform Mobile Toolkit

  • Backend services utilizing OpenShift and Kubernetes
  • Modern App Development (using react, mean, node.js, nosql)
  • Mobile and Web single sourcing
  • Operations => System of record
  • Security

With AeroGear, you can easily:

  • Add notification logic to your app
  • Build native, Cordova, or Xamarin apps automatically
  • Add Google/Facebook/Twitter auth to your app
  • Monitor user statistics of your app

If you are new to AeroGear, we suggest you walk through our Getting Started tutorial first, to get everything set up for development. You can also learn more about each module below.

AeroGear Push

Send push notifications to any device, regardless of platform or network.

Send push notifications from the Unified Push Server via 7 push networks to any mobile device.

Learn more about AeroGear Push

AeroGear Security

Integrate with your existing security and authentication infrastructure.

A small set of classes to provide integration with security providers written in Java, like PicketLink and Apache Shiro.

Learn more about AeroGear Security

AeroGear Digger

Build your mobile apps in the cloud with pipelines.

Build mobile apps in an isolated, scalable and deterministic way in the cloud.
